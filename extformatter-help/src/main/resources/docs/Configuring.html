Configuring External Code Formatter

External Code Formatter is a plugin for integrating Eclipse's code formatter and any command line code formatters into IntelliJ IDEA. When the plugin has been enabled, it will transparently replace IDEA's built-in code formatter. You can reformat your code as before, by pressing Ctrl+Alt+L, or when committing files to VCS. If the external code formatter does not support a file type, or you try to reformat a selected text, the plugin will automatically fall back to using IDEA's code formatter that time.

Using Eclipse's code formatter

  1. In your Project Settings, open External Code Formatter's configuration and tick Enable for current project
  2. Browse for the location of the Eclipse executable. You will need to have Eclipse 3.2 or later installed.
 

On Windows, it is better to select eclipsec.exe instead of eclipse.exe, because the latter one would pop up a black command line dialog during reformatting. However, in both cases Eclipse's startup logo will show up momentarily.

Create Your First Project

  1. On the File menu select New Project to start the New Project wizard.
  2. Walk through the wizard steps to create and configure your project to use a single Java Module. Note that here we're creating the project from scratch, though you can import any other existing sources (Eclipse, Maven, JBuilder, or other code) via this wizard.
    More info about creating projects
  3. Right-click the src directory in the Project view and select New, Class
  4. Type main in the appeared dialog and click OK. IntelliJ IDEA will create the stub file for you.
  5. Type this sample code: 

            
        import java.io.BufferedReader;
        import java.io.IOException;
        import java.io.InputStreamReader;

        public class Main {

            public static void main(String[] argv) throws IOException {
                BufferedReader buff = new BufferedReader(new InputStreamReader(System.in));

                System.out.print("What's your name? ");

                System.out.flush();

                String s = buff.readLine();

                System.out.printf("Hello, %s!", s);

            }
        }
 

Do not copy-paste this code. Try typing it live to experience InteliJ IDEA's advanced code editing features.
  1. On the Run menu click Edit Configurations. Click  and select Application to create the run configuration for your project. Specify the Main Class and click OK.
